Waarom maak ik deze portfolio?

Ik maak deze portfolio voor coderklas. Ik moet dus zelf een website bouwen die over mij gaat. Ik doe dat voor html css 2. Waarom heb ik html css gekozen inplaats van andere modules. Ik vind html leuk. Ik heb het ook al veel gedaan. Eerst begon in nu al 4 jaar geleden om een coder kamp waar ik een website over roblox heb gemaakt. En sinds ik op de middelbare school zit heb ik al html css 0 en 1 voltooid Ik vind het ook leuk om je eigen website te bouwen en daarom heb ik html css 2 als eerst gekozen.
